# Online Course Platform

An **online course platform** web application built with PHP, HTML, CSS, JavaScript, and MySQL.  
This project allows anyone to **browse, enroll in, and manage online courses** — whether you’re a student or an instructor.

It supports user authentication, course browsing, cart/checkout, and student dashboards.  
The platform is suitable for learning and teaching online lessons, making it a full-stack educational system.

---

## 🚀 Features

This online course system includes the following core functionality:

### 📌 Public Features
- 👉 Browse all available courses
- 🔎 View course categories
- 📄 Course detail pages
- 🛒 Shopping cart
- 💳 Checkout process
- 🙋‍♂️ Contact page
- 📜 Terms & privacy pages

### 👨‍🎓 Student Features
- 📥 Register & login
- 📋 View enrolled courses
- 💬 View messages
- ❤️ Wishlist
- 🧾 Order history & invoice
- 🧑‍💼 Profile management
- ⭐ Course reviews

### 👨‍🏫 Admin Features
- 📊 Admin dashboard
- 📚 Manage courses, categories, lessons
- 👤 Manage students & instructors
- 📦 Manage orders
- ✉️ Admin messaging
